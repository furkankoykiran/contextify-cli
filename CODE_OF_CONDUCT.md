# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be kind. Disagree without being disagreeable. Don't be a jerk.

Report unacceptable behavior by opening a confidential GitHub Security Advisory on this repo, or by email to the maintainer listed in [package.json](package.json).
